5. Keyboard shortcuts

5.1. General shortcuts

Shortcut

Action

CTRL-N

Create a new database.

CTRL-O

Open an existing database.

CTRL-Q

Close blunderDB.

CTRL-H, ?

Afficher/cacher l’aide.

5.2. Raccourcis du mode NORMAL

Shortcut

Action

CTRL-I

Import a position from a text file (txt).

CTRL-C

Exporter une position dans le presse-papier en vue d’un import dans XG.

CTRL-V

Import an XG position.

Del

Supprimer la position courante.

TAB

Switch to EDIT mode.

SPACE

Switch to COMMAND mode.

PageUp, h

First position.

GAUCHE, k

Previous position.

DROITE, j

Next position.

PageDown, l

Last position.

CTRL-LEFT

Orientation du board à gauche.

CTRL-RIGHT

Orientation du board à droite.

CTRL-K

Afficher la fenêtre de navigation de positions.

CTRL-L

Show/hide the analysis.

CTRL-P

Afficher/cacher les commentaires.

5.3. Raccourcis du mode EDIT

Shortcut

Action

TAB

Switch to NORMAL mode.

SPACE

Switch to COMMAND mode.

BACKSPACE

Clear the current position.

CTRL-S

Ajouter une position.

CTRL-U

Mettre à jour une position.

CTRL-F

Rechercher une position.

5.4. Raccourcis du mode COMMAND

Shortcut

Action

ENTER

Exécuter une requête.

ESC

Quitter le mode COMMAND.

BACKSPACE

Effacer la commande. Si vide, fermer le mode COMMAND
